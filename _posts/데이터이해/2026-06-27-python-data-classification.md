## 데이터 분류

```
[데이터]

 ├─ 범주형 (라벨·분류)                                      ------- 연산이 불가능
 │   ├─ 명목형 : 순서 X       ex) 성별, 혈액형               ------- 순서가 없음
 │   └─ 서열형 : 순서 O, 간격 X ex) 만족도, 학년              ------- 순서가 있음
 │
 └─ 수치형 (측정값)                                         ------- 연산이 가능
     ├─ 이산형 : 셀 수 있음    ex) 자녀 수                   -------- 소수점으로 쪼갤 수 없음
     └─ 연속형 : 무한 분할 가능 ex) 키, 시간                  -------- 소수점으로 쪼갤 수 있음

```

| 구분 | 하위 분류 | 특징 | 예시 |
|---|---|---|---|
| 범주형 (연산 불가능) | 명목형 | 순서 없음, 구분만 가능 | 성별, 혈액형 |
| 범주형 (연산 불가능) | 서열형 | 순서 있음, 간격 의미 없음 | 만족도, 학년 |
| 수치형 (연산 가능) | 이산형 | 셀 수 있음, 소수점 분할 불가능 | 자녀 수 |
| 수치형 (연산 가능) | 연속형 | 무한 분할 가능, 소수점 분할 가능 | 키, 시간 |

## 척도 (측정 수준)

```text
[척도 (측정 수준)]

 ├─ 질적척도 (범주형)                                        ─────── 수치 연산 불가
 │   ├─ 명목척도 : 단순 분류          ex) 성별, 혈액형         ─────── 구분만 가능 ( = , ≠ )
 │   └─ 서열척도 : 순서 O, 간격 X     ex) 만족도, 석차         ─────── 순위 비교 가능 ( < , > )
 │
 └─ 양적척도 (수치형)                                        ─────── 수치 연산 가능
     ├─ 등간척도 : 간격 O, 절대 0 X   ex) 섭씨온도, IQ         ─────── 곱셈·나눗셈 X / 0이 의미가 있음
     └─ 비율척도 : 간격 O, 절대 0 O   ex) 무게, 시간, 키       ─────── 모든 연산 가능 (+ − × ÷) / 0이 의미가 없음

※ 명목 < 서열 < 등간 < 비율 순으로 정보량이 많아진다
※ 높은 척도는 낮은 척도로 정보를 버리면 변환 가능 (반대는 불가)  ex) 172.5cm → 2등 → 중간

```

| 구분 | 하위 척도 | 가능한 연산 | 예시 |
|---|---|---|---|
| 질적척도 (수치 연산 불가) | 명목척도 | 구분만 가능 ( =, ≠ ) | 성별, 혈액형 |
| 질적척도 (수치 연산 불가) | 서열척도 | 순위 비교 가능 ( <, > ) | 만족도, 석차 |
| 양적척도 (수치 연산 가능) | 등간척도 | 곱셈·나눗셈 불가능, 절대 0 없음 | 섭씨온도, IQ |
| 양적척도 (수치 연산 가능) | 비율척도 | 모든 연산 가능 (+, −, ×, ÷), 절대 0 있음 | 무게, 시간, 키 |

- 정보량 순서는 명목 < 서열 < 등간 < 비율 순으로 많아짐
- 높은 척도는 낮은 척도로 변환 가능하나 반대 방향 변환은 불가능함

## 핵심 요약

- 데이터는 범주형과 수치형으로 나뉨
- 척도는 질적척도와 양적척도로 나뉘며 정보량 차이가 존재함
- 분석 방법 선택 전 데이터의 척도 수준 확인이 필수임
