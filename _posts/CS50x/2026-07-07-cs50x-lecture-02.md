## 0. 개요

- 텍스트 기반 언어인 C 학습
- 단순히 "프로그래밍하는 법"을 배우는 것을 넘어, **좋은 코드를 작성하는 법**까지 함께 배우는 것을 목표로 한다.

## 1. Hello World

[핵심 명령어]

| 명령어 | 역할 |
|---|---|
| `code hello.c` | hello.c라는 새 파일을 생성하고 편집기를 염 |
| `make hello` | hello.c를 컴파일해서 실행 파일 hello를 생성 |
| `./hello` | 컴파일된 프로그램 hello를 실행 |

- `#include <stdio.h>`: 표준 입출력 관련 기능을 사용하겠다는 선언
- `int main(void)`: 프로그램이 실행될 때 가장 먼저 호출되는 함수
- `printf(...)`: 괄호 안의 텍스트를 화면에 출력하는 함수

```c
#include <stdio.h>

int main(void)
{
    printf("hello, world\n");
}
```

```text
hello, world
```

- 컴파일 후 소스 파일(hello.c)과 별개로 실행 파일(hello)이 새로 생성됨

## 2. 이스케이프 문자

| 표기 | 의미 |
|---|---|
| `\n` | 새 줄 생성 |
| `\r` | 줄의 시작으로 이동 |
| `\"` | 큰따옴표 출력 |
| `\'` | 작은따옴표 출력 |
| `\\` | 백슬래시 출력 |

- `\n`은 그 위치에서만 줄바꿈을 수행함 : `%i`나 다른 값 앞뒤에 자동으로 줄이 나눠지는 게 아니라, 문자열 안에 적힌 그대로 동작함

## 3. 사용자 입력

- `get_string` 함수: `cs50.h` 라이브러리의 함수로, 사용자로부터 문자열 입력을 받아 변수에 저장
- `%s` 자리에 변수의 값이 끼워 넣어지는 구조

```c
#include <cs50.h>
#include <stdio.h>

int main(void)
{
    string answer = get_string("이름이 뭐예요? ");
    printf("hello, %s\n", answer);
}
```

```text
이름이 뭐예요? David
hello, David
```

## 4. 조건문

```c
// 예: x=3, y=7일 때
if (x < y)
{
    printf("x는 y보다 작습니다\n");
}
else
{
    printf("x는 y보다 작지 않습니다\n");
}
```

```text
x는 y보다 작습니다
```

- 세 가지 경우(작다/크다/같다)를 모두 다룰 땐 마지막을 `else if`가 아니라 `else`로 처리해 불필요한 비교 제거

## 5. 형식 코드

| 형식 코드 | 대응 자료형 |
|---|---|
| `%c` | char (문자 하나) |
| `%f` | float (실수) |
| `%i` | int (정수) |
| `%li` | long (큰 범위의 정수) |
| `%s` | string (문자열) |

```c
int age = 20;
printf("나이: %i\n", age);

float pi = 3.14;
printf("파이: %f\n", pi);
```

```text
나이: 20
파이: 3.140000
```

## 7. compare.c : 정수 비교 실습

```c
#include <cs50.h>
#include <stdio.h>

int main(void)
{
    int x = get_int("x는 무엇인가요? ");
    int y = get_int("y는 무엇인가요? ");

    if (x < y)
    {
        printf("x는 y보다 작습니다\n");
    }
    else if (x > y)
    {
        printf("x는 y보다 큽니다\n");
    }
    else
    {
        printf("x는 y와 같습니다\n");
    }
}
```

```text
x는 무엇인가요? 10
y는 무엇인가요? 4
x는 y보다 큽니다
```

## 8. 문자열과 문자의 차이 : agree.c

| 자료형 | 의미 |
|---|---|
| string | 여러 문자로 이루어진 문자열 (큰따옴표) |
| char | 문자 하나 (작은따옴표) |

```c
#include <cs50.h>
#include <stdio.h>

int main(void)
{
    char c = get_char("동의하십니까? ");

    if (c == 'y')
    {
        printf("동의함.\n");
    }
    else if (c == 'n')
    {
        printf("동의하지 않음.\n");
    }
}
```

```text
동의하십니까? y
동의함.
```

- 대문자까지 처리하려면 `||`(논리 or) 연산자로 조건을 합침

```c
if (c == 'Y' || c == 'y')
{
    printf("동의함.\n");
}
else
{
    printf("동의하지 않음.\n");
}
```

```text
동의하십니까? Y
동의함.
```

## 9. 반복문 : meow.c

| 반복문 | 특징 |
|---|---|
| `while` | 조건을 먼저 검사 (거짓이면 한 번도 실행 안 될 수 있음) |
| `for` | 초기화·조건·증감을 한 줄에 표현 |
| `do-while` | 먼저 실행 후 조건 검사 (최소 1회 실행 보장) |

```c
for (int i = 0; i < 3; i++)
{
    printf("meow\n");
}
```

```text
meow
meow
meow
```

- 잘못된 입력(음수) 방지: `continue`/`break` 또는 `do-while`로 유효한 값이 들어올 때까지 반복 검증

## 10. 함수 만들기

- `void`는 "반환값 없음", `(void)`는 "입력값 없음"을 의미
- 함수를 아래쪽에 정의해도 위쪽에 프로토타입을 선언하면 먼저 호출 가능
- 매개변수를 추가하면 함수를 더 유연하게 재사용 가능
- 같은 이름의 변수라도 함수마다 독립적인 스코프(유효범위)를 가지며, 값은 복사되어 전달됨

```c
# 함수의 실제 정의(구현부) == return값 없는 def 함수
void meow(int n)
{
    for (int i = 0; i < n; i++)
    {
        printf("meow\n");
    }
}

# 프로그램의 시작점 == if __name__ == "__main__":
# `main`의 반환값은 프로그램 정상 종료 여부(0=성공)를 OS에 알리는 신호라, 이 규약상 정수(int)로 고정
int main(void)
{
    meow(3);
}
```

```text
meow
meow
meow
```

## 11. 마리오 문제 : 중첩 반복문

```c
const int n = 3;
for (int i = 0; i < n; i++)
{
    for (int j = 0; j < n; j++)
    {
        printf("#");
    }
    printf("\n");
}
```

```text
###
###
###
```

- 바깥 반복문(`i`)은 줄, 안쪽 반복문(`j`)은 한 줄의 개수를 담당
- `const`로 고정하면 값이 실수로도 바뀌지 않음
- 반복되는 로직은 헬퍼 함수(`print_row` 등)로 분리 가능

| 연산자 | 의미 |

## 13. 정수 오버플로우

- 자료형이 저장 가능한 값의 한계를 넘으면 값이 예측 불가능하게 뒤집히는 현상
- `int`는 약 21억(부호 있음) ~ 약 42억(부호 없음)이 한계

```text
현재 1073741824달러입니다. 두 배로? y
현재 -2147483648달러입니다.
```

- 한 번의 곱셈으로 최댓값을 넘는 순간, 값이 갑자기 음수로 바뀜

## 14. 자료형 한계와 long

- `int`의 한계를 넘어서는 값은 `long` 타입과 `%li` 형식 코드로 처리
- `long`을 써도 오버플로우가 완전히 사라지는 건 아니고, 한계가 늘어날 뿐

```c
long dollars = 1;
printf("%li\n", dollars);
```

## 15. 형변환과 실수 정밀도

- 정수끼리 나누면 소수점이 잘려나가는 절단(truncation) 발생

```c
printf("%i\n", 7 / 2);
```

```text
3
```

- `(float)`로 형변환하면 소수점을 살릴 수 있음

```c
printf("%f\n", (float) 7 / 2);
```

```text
3.500000
```

- 실수(float)도 이진 표현 방식 때문에 완벽하게 정확한 값을 담지 못함 (`%.50f`로 출력하면 미세 오차가 드러남)
