## 개요

- XGBoost(eXtreme Gradient Boosting)는 그래디언트 부스팅에 **정규화와 병렬 처리**를 추가한 고성능 앙상블 알고리즘임
- Kaggle 등 데이터 사이언스 대회에서 압도적인 성능으로 널리 채택됨

---

## 기본 이론

- 그래디언트 부스팅은 이전 모델의 **잔차(오차)** 를 학습하는 방식으로 순차적으로 트리를 추가함
- XGBoost는 여기에 L1/L2 정규화 항을 추가하고, 2차 미분(Hessian)을 활용해 최적 분할을 빠르게 탐색함

---

## 작동 원리

1. 초기 예측값(평균 등)으로 시작함
2. 현재 예측의 잔차를 계산함
3. 잔차를 학습 목표로 새로운 결정 트리를 학습함
4. 학습률(learning_rate)만큼 트리를 더해 예측을 업데이트함
5. 2~4 과정을 n_estimators 수만큼 반복함

---

## 주요 개념

| 파라미터 | 설명 | 조정 방향 |
|----------|------|----------|
| `n_estimators` | 트리 수 | 많을수록 성능↑, 과적합 위험 |
| `learning_rate` | 학습률 | 작을수록 안정적, 트리 수 필요 |
| `max_depth` | 트리 최대 깊이 | 작을수록 단순 모델 |
| `subsample` | 각 트리에 사용할 샘플 비율 | 0.8~1.0 권장 |
| `colsample_bytree` | 각 트리에 사용할 특성 비율 | 0.5~1.0 권장 |
| `reg_alpha / reg_lambda` | L1/L2 정규화 | 과적합 방지 |

- **조기 종료(Early Stopping)** : 검증 점수가 개선되지 않으면 학습을 자동으로 멈춤

---

## 장단점

**장점**
- 속도가 빠르고 정규화로 과대적합이 잘 방지됨
- 결측치를 자체적으로 처리할 수 있음
- 특성 중요도 제공

**단점**
- 하이퍼파라미터 수가 많아 튜닝이 복잡함
- 매우 대용량 데이터에서는 LightGBM 대비 느릴 수 있음
