## 개요

- 딥러닝은 여러 층(Layer)의 인공 신경망으로 구성된 머신러닝의 한 분야임
- 이미지, 음성, 텍스트 등 비정형 데이터에서 탁월한 성능을 보임

---

## 기본 이론

- **뉴런(Neuron)** : 입력값에 가중치를 곱하고 편향을 더한 뒤 활성화 함수를 통과시킴
- 여러 뉴런이 층을 이루고, 층들이 쌓여 **신경망(Neural Network)** 을 형성함

![신경망 구조](/assets/img/ai-algorithms/neural-network.svg){: .normal}

---

## 작동 원리

### 순전파 (Forward Propagation)
- 입력층 → 은닉층 → 출력층 순서로 값이 전달됨
- 각 층에서 `z = w·x + b` 계산 후 활성화 함수를 적용함

### 역전파 (Backpropagation)
- 출력층에서 계산한 손실(오차)을 역방향으로 전달함
- **편미분(Chain Rule)** 으로 각 가중치에 대한 손실 기울기를 계산함
- 경사하강법으로 가중치를 업데이트함

### 경사하강법 (Gradient Descent)
- 손실 함수의 기울기(gradient) 반대 방향으로 가중치를 조금씩 이동함
- `w := w - α · ∂L/∂w` (α = 학습률)

![경사하강법](/assets/img/ai-algorithms/gradient-descent.svg){: .normal}

## 주요 개념

- **활성화 함수** : 선형 변환에 비선형성을 부여해 복잡한 패턴 학습을 가능하게 함

| 함수 | 수식 | 사용처 |
|------|------|-------|
| ReLU | max(0, x) | 은닉층 기본값 |
| Sigmoid | 1/(1+e⁻ˣ) | 이진 분류 출력층 |
| Softmax | eˣ/Σeˣ | 다중 분류 출력층 |
| Tanh | (eˣ-e⁻ˣ)/(eˣ+e⁻ˣ) | LSTM 등 |

- **배치 크기(Batch Size)** : 한 번에 학습할 데이터 수 — 작을수록 업데이트가 잦고 노이즈가 많음
- **에포크(Epoch)** : 전체 훈련 데이터를 한 번 다 학습한 횟수
- **드롭아웃(Dropout)** : 학습 중 일부 뉴런을 무작위로 비활성화해 과대적합을 방지함
- **배치 정규화(Batch Normalization)** : 각 층의 출력을 정규화해 학습을 안정화함

## 장단점

**장점**
- 비정형 데이터(이미지, 자연어)에서 기존 머신러닝을 크게 뛰어넘는 성능을 냄
- 층을 깊게 쌓을수록 더 복잡한 패턴을 학습함

**단점**
- 대량의 데이터와 GPU 연산이 필요함
- 하이퍼파라미터 튜닝이 복잡하고, 모델 해석이 매우 어려움 (블랙박스)
