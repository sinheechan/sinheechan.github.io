## 개요

- GMM(Gaussian Mixture Model)은 데이터가 여러 개의 **가우시안(정규) 분포의 혼합**으로 이루어져 있다고 가정하는 확률적 군집화 알고리즘임
- K-Means의 하드 할당(Hard Assignment)과 달리, 각 포인트가 여러 군집에 속할 확률을 계산하는 **소프트 클러스터링(Soft Clustering)** 임

---

## 기본 이론

- 데이터 전체가 K개의 가우시안 분포의 합으로 생성된다고 가정함
- 각 가우시안은 **평균(μ)**, **분산·공분산(σ²)**, **혼합 가중치(π)** 세 가지 파라미터로 정의됨
- 모든 가우시안의 혼합 가중치 합은 1임

## 작동 원리 — EM 알고리즘

GMM은 **EM(Expectation-Maximization) 알고리즘** 으로 파라미터를 추정함

1. **초기화** : K개 가우시안의 파라미터를 무작위로 설정함
2. **E 단계(Expectation)** : 현재 파라미터로 각 포인트가 각 군집에 속할 확률(책임도)을 계산함
3. **M 단계(Maximization)** : 계산된 책임도를 이용해 파라미터(μ, σ², π)를 업데이트함
4. 파라미터 변화가 수렴할 때까지 2~3 과정을 반복함

---

## 주요 개념

- **소프트 클러스터링** : 각 포인트가 여러 군집에 속할 확률을 가짐 — K-Means처럼 하나의 군집에 딱 잘라 할당하지 않음
- **공분산 타입(covariance_type)** : 가우시안의 모양을 제어함

| 타입 | 설명 |
|------|------|
| `full` | 각 군집이 자유로운 타원형 — 가장 유연 |
| `tied` | 모든 군집이 같은 모양 |
| `diag` | 축에 정렬된 타원형 |
| `spherical` | 구형 (K-Means와 유사) |

- **BIC / AIC** : 적절한 군집 수 K를 선택하는 데 사용하는 정보 기준 — 낮을수록 좋음

---

## 장단점

**장점**
- 타원형 등 다양한 형태의 군집을 모델링할 수 있음
- 소프트 할당으로 불확실성을 확률로 표현함
- 군집 경계가 겹치는 데이터에 적합함

**단점**
- K-Means보다 계산량이 많음
- K 값을 여전히 지정해야 하며 초기값에 민감함
- 데이터가 가우시안 분포를 따르지 않으면 성능이 저하됨
