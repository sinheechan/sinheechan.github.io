## 개요

- 로지스틱 회귀는 선형 결합값에 **시그모이드 함수**를 씌워 출력값을 0~1 확률로 변환하는 이진 분류 알고리즘임
- 이름은 회귀지만 실제로는 **분류** 문제에 사용됨

---

## 기본 이론

- 선형 회귀의 출력값 `z = w·x + b`를 시그모이드 함수에 통과시킴
- **시그모이드 함수** : σ(z) = 1 / (1 + e⁻ᶻ) — 어떤 값이든 0~1 사이로 압축함
- 출력 확률이 0.5 이상이면 클래스 1, 미만이면 클래스 0으로 분류함

![시그모이드 함수](/assets/img/ai-algorithms/sigmoid.svg){: .normal}

---

## 작동 원리

1. 입력 특성에 가중치를 곱한 선형 결합값 z를 계산함
2. z를 시그모이드 함수에 통과시켜 확률값 p를 출력함
3. **크로스 엔트로피 손실(Log Loss)**을 최소화하는 방향으로 가중치를 업데이트함
4. 경사하강법(Gradient Descent)으로 최적 가중치를 탐색함

---

## 주요 개념

- **크로스 엔트로피 손실** : 예측 확률과 실제 레이블의 차이를 측정하는 손실 함수 — MSE 대신 사용하는 이유는 확률 기반 출력에 더 적합하기 때문
- **결정 임계값(Threshold)** : 기본 0.5이며, 상황에 따라 조정 가능함 (예: 암 진단은 0.3으로 낮춰 민감도↑)
- **다중 클래스** : `softmax` 함수와 조합해 3개 이상 클래스도 처리 가능함

---

## 장단점

**장점**
- 모델이 단순하고 빠르며 확률값으로 결과를 해석할 수 있음
- 계수(가중치)를 통해 각 특성의 영향을 파악할 수 있음

**단점**
- 선형 결정 경계만 표현 가능해 비선형 데이터에 한계가 있음
- 특성 간 다중공선성이 있으면 성능이 저하됨

---

## 평가지표

- **정확도(Accuracy)** : 전체 중 맞게 분류한 비율
- **정밀도(Precision)** : 양성으로 예측한 것 중 실제 양성 비율
- **재현율(Recall)** : 실제 양성 중 양성으로 맞게 예측한 비율
- **F1-Score** : 정밀도와 재현율의 조화평균
- **ROC-AUC** : 임계값 변화에 따른 분류 성능의 종합 지표
